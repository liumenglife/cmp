# 合同分类与字段设计历史参考（2026-03-31）

本文档仅作为历史参考留痕，不是当前正式基线，也不作为当前执行依据。

## 1. 分类层级

| 层级 | 内容 |
|---|---|
| 文书形态 | 合同、协议、框架 |
| 业务域 | 销售、采购、HR |
| 子类 | 见下表 |
| 标签 | 仅补充说明，不驱动模板 |

| 业务域 | 子类 |
|---|---|
| 销售 | 主营业务销售合同、其他业务销售合同 |
| 采购 | 生产性采购合同、零星采购合同 |
| 框架 | 销售框架协议、采购框架协议 |
| HR | 劳动合同、保密协议、竞业限制协议、培训服务期协议、离职/解除协议 |

| 子类 | 细分 |
|---|---|
| 主营业务销售合同 | 主机销售、配件销售 |
| 其他业务销售合同 | 废旧物资销售、残值物销售、边角料/报废物资销售、危废/特殊处置类销售 |
| 生产性采购合同 | 原材料、核心零部件、外协加工、工装模具、研发试制、包装辅材 |
| 零星采购合同 | 办公、后勤、员工福利、IT服务、咨询服务 |
| 销售框架协议 | 主营业务销售框架、配件销售框架、其他业务销售框架 |
| 采购框架协议 | 生产性采购框架、零星采购框架 |

## 5. 子类特殊字段

| 子类 | 特殊字段 |
|---|---|
| 主机销售 | 产品型号、配置清单、安装调试、培训、整机验收、试运行、质保期、售后响应、备件支持、项目里程碑 |
| 配件销售 | 料号、适配机型、规格型号、最小起订量、批次供货、价格梯度、快速发货、备件替换、库存支持、配件质保 |
| 其他业务销售 | 物资来源、处置方式、计量/过磅规则、环保合规、危废资质、提货/清运、风险转移时点、特殊税务处理 |
| 生产性采购合同 | 物料/零部件描述、规格型号、技术参数、质量标准、首件确认、批次交付、包装要求、追溯要求、质保金、知识产权归属 |
| 零星采购-办公 | 采购清单、数量、单价、交付地址、简化验收、月结/次月结算、紧急交付、收货确认 |
| 零星采购-后勤 | 采购清单、服务范围、数量、单价、交付/履约地址、简化验收、月结/次月结算、收货确认 |
| 零星采购-员工福利 | 福利物资范围、发放对象、预算、发放方式、签收/领用确认 |
| 零星采购-IT服务 | 服务范围、交付物、服务周期、SLA、账号/权限交付、安全要求、源码/账号归属、维护响应 |
| 零星采购-咨询服务 | 咨询范围、交付报告、里程碑、顾问名单、验收标准、保密、知识产权归属 |
| 销售框架协议 | 年度预测量、价格体系、折扣/返点、价格保护、最惠条款、订单释放规则、区域范围、排他/独家、结算周期 |
| 采购框架协议 | 供货能力、价格锁定周期、采购预测、订单释放规则、年度对账、供货优先级、质量协定、SLA |
| 劳动合同 | 试用期、岗位职责、考核规则、工作时间、薪酬、绩效、奖金、社保、公积金、加班、休假、培训、交接 |
| 保密协议 | 保密范围、保密期限、资料返还、违约责任、信息使用限制 |
| 竞业限制协议 | 竞业期限、竞业地域、限制对象、补偿标准、违约金、离职后义务 |
| 培训服务期协议 | 培训内容、培训费用、服务期限、违约返还、提前离职责任 |
| 离职/解除协议 | 解除原因、工资结算、补偿、交接清单、权限回收、保密延续、竞业衔接、无争议确认 |
